# WifiStatusBar
Instantly change the Wifi Status Icon on the bar Hackintosh
=============================================================

Open EFI folder

Copy kext (driver) file to EFI/OC/Kext

#  Install
Open Terminal 

$ ./install.sh

# Run

$ WifiStatusBar
